## Getting started

1. **Установите зависимости**:
   ```bash
   npm i
   ```

2. **Переименуйте .env.example на .env, и вставьте API keys**:
   ```bash
   cp .env.example .env
   ```
   ```bash
    GEMINI_API_KEY=your_api_key_here
   ```

3. **Run locally**:
   ```bash
   npm run dev
   ```

4. **Build check**:
   ```bash
   npm run build
   ```
